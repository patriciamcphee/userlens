## üé® Logo Design

### Design Philosophy
The UserLens logo combines a **user icon** with a **lens metaphor**, creating a powerful visual representation of our mission to provide clear user insights. The forward-pointing negative space suggests progress and forward-thinking approaches to user research.

### Logo Elements
1. **User Icon**: Simplified human figure (circle + curved body)
2. **Circular Lens**: Sweeping curves creating a lens/eye effect
3. **Directional Arrow**: Negative space pointing right (‚Üí insight & progress)
4. **Modern Geometry**: Clean, scalable shapes for digital excellence

### 1. Primary Logo (Full)

- **Use**: Favicons, app icons, social media profiles

- **Size**: Square format, minimum 16px

### 3. Wordmark Only
- **Use**: Text-heavy contexts, secondary branding
- **Context**: Email signatures, co-branding scenarios
- **Includes**: "UserLens" text + optional tagline

### 4. Horizontal Layout
- **Use**: Headers, navigation bars, landscape orientations
- **Context**: Wide format spaces
- **Alignment**: Icon left, text right

### 5. Vertical Layout
- **Use**: Splash screens, mobile apps, square formats
- **Context**: Tall or square containers
- **Alignment**: Icon top, text bottom

## ‚úÖ Logo Do's

### ‚úÖ Correct Usage
- Use official logo files only
- Maintain proper clear space
- Ensure sufficient contrast with backgrounds
- Use appropriate size for context
- Maintain aspect ratio when scaling
- Use dark mode version on dark backgrounds
- Place on neutral or complementary backgrounds

### ‚úÖ Preferred Backgrounds
- Pure white or light gray
- Dark charcoal or pure black (dark mode)
- Subtle gradients that don't interfere
- Photographs with clear, uncluttered areas

- ‚ùå Don't use low-resolution versions
